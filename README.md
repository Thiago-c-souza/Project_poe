# Mini ARPG Elemental

Protótipo web simples inspirado na ideia de um PoE simplificado. Crie um personagem a partir de três classes coloridas, aloque pontos em uma árvore passiva compacta e enfrente três mapas curtos com mobs normais, mágicos, raros e um chefe.

## Como rodar

1. Instale uma extensão HTTP simples ou use o Python já disponível.
2. No diretório do projeto, inicie um servidor local:

```bash
python -m http.server 8000
```

3. Acesse no navegador: `http://localhost:8000` e interaja com a página.

Nenhuma dependência adicional é necessária, os arquivos HTML/CSS/JS já entregam todo o protótipo.
